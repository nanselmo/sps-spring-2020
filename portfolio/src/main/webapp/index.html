Middle School Robotics Team (Past)

Prior to joining Google, I was a developer at Cesar Chavez Elementary School. I started a Code Club and Robotics Team. In our first year, we were the only Chicago Public School team to make it to the state and then national competition. Check us out on Chicago's local news station here.

Weasley Clock (Future)

I don't make time to tinker as much as I should, but I would love for my next project to be this Weasley Clock. I would use Google Maps to determine my location (and my partner's) and ping that to an Arduino which would use servo motors to update a clock. There are some instructions on the web including these from Instructables and these from Adafruit.
